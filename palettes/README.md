Palette Library

Drop palette files here. Subfolders become categories in the UI.

Supported formats:
- .ase (Adobe Swatch Exchange)
- .gpl (GIMP palette)
- .hex or .txt (one hex color per line, e.g. FF00AA)

Notes
- The app will also create palettes\Imported when you use the Import Palette button.
- Files in the root of palettes/ show up under the "Local" category.
